# Cheatsheet: Exploratory Data Analysis (EDA) with Python

## Overview
This cheatsheet provides a quick reference for performing exploratory data analysis (EDA) using Python. It includes key concepts, tools, and practical tips for understanding and visualizing data.

## Key Concepts

### 1. Exploratory Data Analysis (EDA)
- **EDA**: The process of analyzing and summarizing data to uncover patterns, trends, and insights.
- **Descriptive Statistics**: Summarize data using measures like mean, median, mode, and standard deviation.
- **Data Visualization**: Use plots and charts to understand data distribution and relationships.
- **Data Distributions**: Understand how data is distributed across different variables.
- **Correlation Analysis**: Identify relationships between variables using correlation coefficients.

### 2. Tools and Libraries
- **Pandas**: For data manipulation and analysis.
- **NumPy**: For numerical computing.
- **Matplotlib and Seaborn**: For data visualization.
- **Seaborn**: For advanced statistical visualizations.

## Practical Tips

### 1. Descriptive Statistics
- Use `df.describe()` to get statistical summaries of the data.
- Use `df.info()` to get information about the DataFrame.

### 2. Data Visualization
- Use `matplotlib.pyplot.plot()` for basic plots.
- Use `seaborn.lineplot()` for line plots.
- Use `seaborn.scatterplot()` for scatter plots.
- Use `seaborn.histplot()` for histograms.
- Use `seaborn.boxplot()` for box plots.
- Use `seaborn.pairplot()` for pairwise relationships.

### 3. Correlation Analysis
- Use `df.corr()` to calculate correlation coefficients between variables.
- Use `seaborn.heatmap()` to visualize correlation matrices.

### 4. Data Distribution
- Use `seaborn.kdeplot()` for kernel density estimates.
- Use `seaborn.distplot()` for distribution plots.
